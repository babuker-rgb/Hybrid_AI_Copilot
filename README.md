# Hybrid AI · Multi-Objective Tablet Optimization

## 🧬 Overview
This framework combines physics-informed neural networks with multi-objective optimization for pharmaceutical tablet formulation.

### Version: v29.28-R32
### Institution: Nile Valley University, Sudan

## 🚀 Features
- **Hybrid AI Model**: Physics-informed neural network with constraints
- **Multi-Objective Optimization**: NSGA-II with Pareto front analysis
- **Interactive Dashboard**: Streamlit-based GUI
- **Docker Support**: Containerized deployment
- **Scalable Architecture**: Support for large-scale optimization
